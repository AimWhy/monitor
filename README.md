## 功能特性
- [x] 🔨监听请求错误
- [x] 🔨console
- [x] 🔨路由跳转
- [x] 🔨代码报错
- [x] 🔨click
- [x] 🔨资源加载错误
- [x] 🏅自定义上报错误
- [x] 🚀丰富的hooks和配置项
- [x] 🌝支持IE8以上
- [x] 👌持续迭代与更新
- [x] 支持react
- [x] 完善的e2e测试
- [ ] 完善的单元测试
- [ ] 支持小程序框架
